What is the project

We have built a system that takes a voice and speaks back in the same voice, almost!

Text to speech systems have conventionally been around for a while now, but always use parameters usch as pitch, frequency, etc. to generate voice using the in-built system processor. But with increasing demand for Human-Computer-Interaction(HCI), custom voices are the necessity. We have tried a basic version of this system, which can be explained in the four following steps.

Get the voice and
it's transcript

The user provides the system with a sample voice file and its corresponding text transcript.

Forced Alignment

The system runs forced alignment(an ML technique) to align and get the timings of the words, and the corresponding sounds that constitute the word.

Making the Database

Based on the timings that are found, the sounds are extracted and stored in a directory. Usually one directory for one speaker.

Synthesize

When the TTS part begins, the input text is split into its corresponding diphones. The required diphones are extracted from the directory and played back in a sequence from the directory.
